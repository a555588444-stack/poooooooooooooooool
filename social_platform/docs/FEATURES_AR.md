# ميزات فايسلووك (مشابهة لـ Facebook + Discord)

## 1) الحسابات
- تسجيل حساب جديد (email/phone/oauth/passkeys).
- تسجيل دخول آمن مع جلسات متعددة الأجهزة.
- إعدادات خصوصية متقدمة + 2FA.

## 2) الميزات الاجتماعية (Facebook-like)
- ملف شخصي، أصدقاء، متابعة، حظر.
- منشورات، تعليقات متداخلة، تفاعلات، Mentions، Hashtags.
- Stories وReels وMarketplace.
- صفحات (Pages) ومجموعات (Groups) وأحداث (Events).

## 3) ميزات المجتمع والدردشة (Discord-like)
- Servers متعددة لكل مجتمع.
- Channels نصية وصوتية.
- Roles وصلاحيات دقيقة.
- رسائل فورية + عداد غير مقروء + إشعارات مباشرة.

## 4) الأعمال والإعلانات
- حسابات إعلانية وحملات وإعلانات.
- قياسات يومية: impressions/clicks/spend/conversions.

## 5) الإشراف والأمان
- نظام بلاغات وإجراءات Moderation.
- سجلات أمنية ومراقبة الدخول.

## 6) التحليلات والتوسع
- Views وMaterialized Views للإحصاءات.
- تقسيم جداول ضخمة (Partitioning) + Sharding metadata.
